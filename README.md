# 🐍 Python Buddy AI

An AI-powered Python learning platform built with Streamlit.

## Features

- 🤖 AI Python Tutor
- 💻 Interactive Code Practice
- 🐞 Debugging Challenges
- 🧠 Python Quizzes
- 🏆 Certificate Generator
- 🔐 Login System
- 📊 Learning Dashboard

## Tech Stack

- Python
- Streamlit
- OpenAI API
- Pillow
- SQLite

## Run Locally

pip install -r requirements.txt

streamlit run python_buddy.py
